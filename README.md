# bibliothèque dev web


mon historique de bloc de code,
qui me permet de réunir en un endroit tout le code que je crée,
pour mes projets pro ou perso en web


## Fonctionnalités

### Calcul simple
- Addition automatique de deux nombres
- Affichage du résultat dans la page

### Interaction utilisateur
- Champ input pour entrer un prénom
- Affichage dynamique du prénom
- Bouton de validation

### Appel API
- Récupération de l’heure en temps réel via API
- Affichage dynamique dans la page

### Visuel et style
- Bulles interactives avec hover et clic
- Boutons et logos animés
- Footer avec liens vers réseaux sociaux


## Technologies utilisées

- HTML
- CSS
- JavaScript
- API WorldTimeAPI


## Ce que j’ai appris

- Manipuler le DOM avec JavaScript
- Récupérer et afficher des données depuis une API
- Gérer des événements utilisateur (click, input)
- Convertir et utiliser des données dynamiques
